# Windows 优化助手 UI 设计规范

## 1. 颜色方案

### 主色调
- 主题蓝色: #0088FF
- 主题蓝色渐变: linear-gradient(180deg, #4BB4FF 0%, rgba(115, 192, 255, 0) 100%)
- 浅蓝色: #64B5F6
- 深蓝色: #1E88E5
- 天蓝色: #36A5FF

### 背景色
- 主背景: #F7F7F8
- 景: #FFFFFF
- 卡片背景: #FFFFFF

### 文字颜色
- 主文字: rgba(0, 0, 0, 0.9)
- 次要文字: rgba(0, 0, 0, 0.6)
- 白色文字: #FFFFFF
- 半透明白色: rgba(255, 255, 255, 0.7)

### 功能色
- 成功/绿色: #28C676, #00B689
- 警告/橙色: #FF6B00
- 错误/红色: #FF3737, #EB5F58
- 紫色: #772DC9

### 辅助色
- 边框灰: #E5E7EB
- 分割线灰: #D8D8D8
- 浅灰背景: rgba(0, 0, 0, 0.05)

### 功能卡片背景色（带透明度）
- 蓝色卡片: rgba(0, 60, 255, 0.05)
- 绿色卡片: rgba(40, 198, 118, 0.1)
- 橙色卡片: rgba(255, 115, 0, 0.1)
- 红色卡片: rgba(255, 0, 0, 0.05)
- 紫色卡片: rgba(114, 0, 255, 0.1)
- 黄色卡片: rgba(254, 187, 0, 0.12)

## 2. 字体规范

### 字体家族
- 主字体: Source Han Sans (思源黑体)
- 备用字体: IKEA Sans

### 字号和字重
- 标题大: 16px Bold (line-height: 16px)
- 标题中: 14px Bold (line-height: 14px)
- 正文: 12px Regular (line-height: 14px)
- 小字: 12px Bold (line-height: 12px)

## 3. 圆角规范
- 卡片圆角: 8px
- 按钮圆角: 4px
- 小元素圆角: 2px

## 4. 阴影效果
- 卡片阴影: box-shadow: 1px 2px 4px 0px rgba(0, 0, 0, 0.1)
- 悬浮阴影: box-shadow: 1.44px 2.89px 5.78px 0px rgba(0, 0, 0, 0.1)
- 功能卡片阴影（带颜色）:
  - 蓝色: box-shadow: 1.23px 2.46px 2.46px 0px rgba(0, 48, 130, 0.2)
  - 绿色: box-shadow: 1.23px 2.46px 2.46px 0px rgba(0, 130, 68, 0.2)
  - 橙色: box-shadow: 1.23px 2.46px 2.46px 0px rgba(168, 65, 0, 0.2)
  - 红色: box-shadow: 1.23px 2.46px 2.46px 0px rgba(130, 0, 0, 0.2)

## 5. 间距规范
- 页面边距: 24px
- 卡片间距: 16px
- 元素间距: 8px
- 小间距: 4px

## 6. 组件尺寸

### 头部导航栏
- 高度: 48px
- 按钮尺寸: 32x32px

### 状态球
- 直径: 120px
- 内圈: 100px

### 功能卡片
- 宽度: 自适应（3列布局）
- 高度: 80px
- 内边距: 16px

### 按钮
- 大按钮: 120x40px
- 中按钮: 100x32px
- 小按钮: 80x28px

## 7. 布局规范

### 主容器
- 最大宽度: 1200px
- 最小宽度: 800px
- 居中对齐

### 网格系统
- 3列布局（功能卡片）
- 列间距: 16px
- 行间距: 16px

## 8. Chrome 69 兼容性注意事项
- 不使用 CSS Grid gap
- 不使用 Flex gap
- 使用 margin 实现间距
- 使用 calc() 计算宽度
- 使用 inline style 设置样式
